Frequently Asked Questions

To enable location services for GeoKR:
  • iPhone (Safari):
    1. Go to Settings > Privacy & Security > Location Services and make sure Location Services are On.
    2. Scroll down to Safari Websites and set to Ask or While Using the App.
    3. If you previously denied location, go to Settings > Safari > Advanced > Website Data, find this site, and remove its data.
    4. Reload the GeoKR page and allow location access when prompted.
  • Android (Chrome):
    1. Go to Settings > Location and ensure Location is On.
    2. In Chrome, tap the lock icon or info icon in the address bar, then tap Permissions and allow Location.
    3. If you previously denied location, clear site settings in Chrome and reload the page.
    4. Allow location access when prompted.
  • General Tips:
    • Make sure you are accessing GeoKR over HTTPS (not HTTP).
    • Some browsers require a user action (like pressing a button) to request your location.
    • If you still have trouble, try restarting your phone after changing settings.
